# Pr3Matem — транспортная задача

В репозитории две реализации метода северо-западного угла:

1. **Консольный скрипт** `transport_solver.py` — интерактивный ввод, балансировка системы и анимация шагов в терминале.
2. **Веб-версия** (`index.html`, `style.css`, `app.js`) — одностраничное приложение с генерацией случайных данных, ручным редактированием тарифов, визуализацией результата и кнопкой «Оптимизировать план» для улучшения опорного решения. После оптимизации отображается пошаговый разбор: потенциалы `u`/`v`, таблица Δ-оценок, визуализация цикла перераспределения (с чередованием знаков) и итоговый текстовый ответ.

## Как запустить сайт

```bash
python3 -m http.server 8000
```

После запуска откройте в браузере `http://localhost:8000/index.html`. Можно просто открыть файл `index.html`, но встроенный сервер гарантирует корректное подключение шрифтов. Сначала нажмите «Рассчитать план», а затем «Оптимизировать план», чтобы применить метод потенциалов к найденному опорному решению, увидеть подробный журнал (потенциалы → Δ → цикл → θ → перераспределение) и получить текстовый ответ с итоговой стоимостью.

## Как запустить консольный решатель

```bash
python3 transport_solver.py
```
