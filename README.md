# Pr3Matem — транспортная задача

В репозитории две реализации метода северо-западного угла:

1. **Консольный скрипт** `transport_solver.py` — интерактивный ввод, балансировка системы и анимация шагов в терминале.
2. **Веб-версия** (`index.html`, `style.css`, `app.js`) — одностраничное приложение с генерацией случайных данных, ручным редактированием тарифов и визуализацией результата.

## Как запустить сайт

```bash
python3 -m http.server 8000
```

После запуска открыть в браузере `http://localhost:8000/index.html`. Можно просто открыть файл `index.html`, но встроенный сервер гарантирует корректное подключение шрифтов.

## Как запустить консольный решатель

```bash
python3 transport_solver.py
```
